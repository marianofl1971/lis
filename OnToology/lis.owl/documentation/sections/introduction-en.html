Introduction back to ToC

The ontology describes the schema followed by the JSON messages exchanged with the [LAS service](https://marianofl1971@bitbucket.org/marianofl1971/mediatorkbservice.git). The input matches [experiment](#Experiment) class. The output is a set of elements matching the [certainty level assignment with explanations](#CLAwithExplanations) class. The examples in JSON have been validated using [JSONLint](https://jsonlint.com/), and the examples in JSON-LD, using [JSON-LD Playground](https://json-ld.org/playground/).
